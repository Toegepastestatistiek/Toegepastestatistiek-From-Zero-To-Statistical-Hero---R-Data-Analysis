Scheefheid en Kurtosis Uitgelegd

Scheefheid en kurtosis beschrijven de vorm van de verdeling. Ze zijn belangrijk voor het beoordelen van normaliteit en de keuze van statistische technieken.

Kurtosis (Gepiektheid)

Definitie

Kurtosis meet de "zwaarte" van de staarten en de "gepiektheid" van de verdeling vergeleken met de normale verdeling.

  • Symbool: Î³â‚‚ (gamma) of Ku
  • Normale verdeling: Kurtosis = 0 (excess kurtosis) of 3 (kurtosis)

Let op: Twee definities!

  • Kurtosis: Normaal = 3
  • Excess kurtosis: Normaal = 0 (= kurtosis - 3)

R's psych::kurtosi() geeft excess kurtosis (normaal = 0).

R Code

# Scheefheid en Kurtosis met psych package
library(psych)

stress <- wave1_coded$W1_Relstress1_Gem

# Scheefheid
scheefheid <- psych::skew(stress, na.rm = TRUE)
cat("Scheefheid:", round(scheefheid, 3), "\n")

# Kurtosis (excess kurtosis in psych)
kurtosis <- psych::kurtosi(stress, na.rm = TRUE)
cat("Kurtosis:", round(kurtosis, 3), "\n")

# Interpretatie
interpret_skew <- function(sk) {
  if (abs(sk) < 0.5) return("Symmetrisch")
  else if (abs(sk) < 1) return("Matig scheef")
  else return("Sterk scheef")
}

interpret_kurt <- function(ku) {
  if (ku > 1) return("Leptokurtisch (zware staarten)")
  else if (ku < -1) return("Platykurtisch (lichte staarten)")
  else return("Mesokurtisch (normaal)")
}

cat("Interpretatie scheefheid:", interpret_skew(scheefheid), "\n")
cat("Interpretatie kurtosis:", interpret_kurt(kurtosis), "\n")

Combineren voor Normaliteitsbeoordeling

Vuistregel voor Normaliteit

Een verdeling is bij benadering normaal als:

  • Scheefheid tussen -1 en +1
  • Kurtosis tussen -1 en +1
  • Shapiro-Wilk p > 0.05 (formele toets)

Belangrijk: Bij n > 100 is de Centrale Limietstelling meestal voldoende voor robuustheid van parametrische toetsen.
