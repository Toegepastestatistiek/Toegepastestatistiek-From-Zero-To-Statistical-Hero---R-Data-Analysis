Wanneer Welke Gebruiken?

Gebruik het GEMIDDELDE wanneer:

  • âœ… De verdeling symmetrisch is
  • âœ… Er geen uitschieters zijn
  • âœ… Je interval- of ratiodata hebt
  • âœ… Je verder wilt rekenen (bijv. t-toets)

Gebruik de MEDIAAN wanneer:

  • âœ… De verdeling scheef is
  • âœ… Er uitschieters zijn
  • âœ… Je ordinale data hebt
  • âœ… Je "typische" waarde wilt rapporteren (bijv. inkomen)

R Code Vergelijking

# Beide maten berekenen
gemiddelde <- mean(data$variabele, na.rm = TRUE)
mediaan <- median(data$variabele, na.rm = TRUE)

# Vergelijk
cat("Gemiddelde:", round(gemiddelde, 2), "\n")
cat("Mediaan:", mediaan, "\n")
cat("Verschil:", round(gemiddelde - mediaan, 2), "\n")

# Interpretatie van scheefheid
if (abs(gemiddelde - mediaan) < 0.1 * sd(data$variabele, na.rm = TRUE)) {
  cat("â†’ Verdeling lijkt symmetrisch\n")
} else if (gemiddelde > mediaan) {
  cat("â†’ Verdeling is rechts scheef (positief)\n")
} else {
  cat("â†’ Verdeling is links scheef (negatief)\n")
}
