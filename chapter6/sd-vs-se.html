De standaardafwijking (SD) en standaardfout (SE) worden vaak verward, maar meten fundamenteel verschillende dingen. Dit onderscheid is essentieel voor correcte statistische rapportage!

Standaardafwijking (SD)

Wat is SD?

De standaardafwijking meet hoe ver de individuele datapunten gemiddeld van het gemiddelde afwijken.

  • Kleine SD â†’ Data dicht bij gemiddelde (homogeen)
  • Grote SD â†’ Data ver uitgespreid (heterogeen)

Voorbeeld: Twee klassen met zelfde gemiddelde

Klas A: Scores 68, 70, 70, 72 â†’ M = 70, SD = 1.63

Klas B: Scores 50, 60, 80, 90 â†’ M = 70, SD = 18.26

Interpretatie: Beide klassen hebben gemiddeld 70, maar Klas B heeft veel meer variatie in prestaties.

Standaardfout (SE)

Wat is SE?

De standaardfout meet de precisie waarmee het steekproefgemiddelde de populatie schat. Het is de standaardafwijking van de steekproevenverdeling.

  • Kleine SE â†’ Nauwkeurige schatting van Î¼

R Code

sd-vs-se-berekening R 
# =============================================
# SD en SE berekenen - Stap voor stap
# =============================================

# Selecteer de variabele
stress <- wave1_coded$W1_Relstress1_Gem

# Stap 1: Bereken basis statistieken
n <- sum(!is.na(stress))          # Aantal geldige observaties
gemiddelde <- mean(stress, na.rm = TRUE)  # Gemiddelde
sd <- sd(stress, na.rm = TRUE)          # Standaardafwijking

# Stap 2: Bereken SE (standaardfout)
se <- sd / sqrt(n)                 # SE = SD / âˆšn

# Stap 3: Rapporteer resultaten
cat("N =", n, "\n")
cat("M =", round(gemiddelde, 2), "\n")
cat("SD =", round(sd, 2), "\n")
cat("SE =", round(se, 3), "\n")

APA Rapportage

Met SD (descriptief):

"Relatiestress had een gemiddelde van 2.16 (SD = 0.93)."

Met SE (inferentieel):

"Relatiestress (M = 2.16, SE = 0.05) verschilde significant..."

Met 95% BI (beste praktijk):

"De gemiddelde relatiestress was 2.16, 95% CI [2.06, 2.26]."

Welke rapporteren in je paper?

  • Resultaten van toetsen: Gebruik SE of 95% BI
