Standaardafwijking (SD)

Wat is SD?

De standaardafwijking meet hoe ver de individuele datapunten gemiddeld van het gemiddelde afwijken.

  • Kleine SD â†’ Data dicht bij gemiddelde (homogeen)
  • Grote SD â†’ Data ver uitgespreid (heterogeen)

Voorbeeld: Twee klassen met zelfde gemiddelde

Klas A: Scores 68, 70, 70, 72 â†’ M = 70, SD = 1.63

Klas B: Scores 50, 60, 80, 90 â†’ M = 70, SD = 18.26

Interpretatie: Beide klassen hebben gemiddeld 70, maar Klas B heeft veel meer variatie in prestaties.

R Code

# SD en SE berekenen
stress <- wave1_coded$W1_Relstress1_Gem

# Basis statistieken
n <- sum(!is.na(stress))
gemiddelde <- mean(stress, na.rm = TRUE)
sd <- sd(stress, na.rm = TRUE)
se <- sd / sqrt(n)

# Rapporteer
cat("N =", n, "\n")
cat("M =", round(gemiddelde, 2), "\n")
cat("SD =", round(sd, 2), "\n")
cat("SE =", round(se, 3), "\n")

# 95% Betrouwbaarheidsinterval (gebaseerd op SE)
ci_lower <- gemiddelde - 1.96 * se
ci_upper <- gemiddelde + 1.96 * se
cat("95% BI: [", round(ci_lower, 2), ",", round(ci_upper, 2), "]\n")

APA Rapportage

Met SD (descriptief):

"Relatiestress had een gemiddelde van 2.45 (SD = 0.92)."

Met SE (inferentieel):

"Relatiestress (M = 2.45, SE = 0.053) verschilde significant..."

Met 95% BI (beste praktijk):

"De gemiddelde relatiestress was 2.45, 95% CI [2.35, 2.55]."
