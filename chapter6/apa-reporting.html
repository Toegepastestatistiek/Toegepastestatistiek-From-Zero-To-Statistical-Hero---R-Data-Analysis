Rapporteren volgens APA 7e Editie

Hoe rapporteer je beschrijvende statistieken professioneel volgens de richtlijnen van de American Psychological Association (APA, 7e editie)?

Basisregels APA Rapportage

Kernprincipes

  1. Symbolen cursief: M, SD, SE, p, t, F, r (NIET cursief: n, df, N)
  2. Spaties rond symbolen: M = 3.45 (niet M=3.45)
  3. Consistente decimalen: Meestal 2 decimalen voor M, SD, SE
  4. Nullen voor decimalen: p = .03 (niet p = 0.03) als waarde < 1
  5. Hoofdletters voor populatie: N (populatie), n (steekproef)

Beschrijvende Statistieken

Gemiddelde en Standaarddeviatie

Formaat: M = waarde, SD = waarde

✅ Correct:

  • De gemiddelde relatiestress was M = 3.45, SD = 0.82.
  • Werkstress (M = 2.87, SD = 1.12) was lager dan relatiestress.
  • Respondenten rapporteerden matige financiële stress (M = 3.21, SD = 0.95).

❌ Fout:

  • M=3.45, SD=0.82 (geen spaties, niet cursief)
  • Mean = 3.45 (gebruik M, niet Mean)
  • M = 3.4500 (te veel decimalen)

Met Betrouwbaarheidsintervallen

Formaat: M = waarde, SD = waarde, 95% BI [ondergrens, bovengrens]

Voorbeelden:

  • Relatiestress: M = 3.45, SD = 0.82, 95% BI [3.32, 3.58]
  • De gemiddelde werkstress was M = 2.87 (SD = 1.12, 95% BI [2.69, 3.05]).

Mediaan en IQR (bij scheefheid)

Formaat: Mdn = waarde, IQR = waarde

Voorbeelden:

  • Het mediane inkomen was Mdn = €35,000, IQR = €18,000.
  • De verdelingen waren rechts-scheef, daarom rapporteren we medianen (Mdn = 3.00, IQR = 2.00).

Afrondingsregels

Statistiek Decimalen Voorbeeld
M, SD, SE, Mdn, IQR 2 decimalen M = 3.45, SD = 0.82
p-waarde (p ≥ .001) 2-3 decimalen p = .03 of p = .003
p-waarde (p < .001) - p < .001 (niet p=.000)
Correlatie (r) 2 decimalen r = .45
Cohen's d 2 decimalen d = 0.52
t, F, χ² 2 decimalen t = 3.21, F = 5.67
df (vrijheidsgraden) 0 decimalen (geheel getal) df = 148
N, n (steekproefgrootte) 0 decimalen (geheel getal) N = 150, n = 75

⚠️ Belangrijke Afrondingsregels

  • Geen leading zero bij p, r, α: p = .03 (niet p = 0.03) omdat deze waarden niet groter dan 1 kunnen zijn
  • Wel leading zero bij M, SD, d: M = 0.45 (wel 0, want waarde kan > 1 zijn)
  • p < .001 in plaats van p=.000: Rapporteer zeer kleine p-waarden als "p < .001"

Tabellen volgens APA

Basisstructuur

APA Tabelregels:

  • Alleen horizontale lijnen (geen verticale lijnen)
  • Dubbele lijn boven en onder de tabel
  • Enkele lijn onder kolomkoppen
  • Titel boven de tabel, cursief
  • Noten onder de tabel indien nodig

APA Rapportage Functie in R

apa_rapporteer <- function(x, naam = "Variable", decimals = 2) {
  # Bereken statistieken
  m <- mean(x, na.rm = TRUE)
  sd_val <- sd(x, na.rm = TRUE)
  n <- sum(!is.na(x))
  
  # 95% BI
  ci <- t.test(x)$conf.int
  
  # Formatteer volgens APA
  m_str <- format(round(m, decimals), nsmall = decimals)
  sd_str <- format(round(sd_val, decimals), nsmall = decimals)
  ci_str <- paste0("[", format(round(ci[1], decimals), nsmall = decimals),
                   ", ", format(round(ci[2], decimals), nsmall = decimals), "]")
  
  # Output
  cat(naam, ": M = ", m_str, ", SD = ", sd_str, 
      ", 95% BI ", ci_str, ", n = ", n, sep = "")
}

# Gebruik
apa_rapporteer(data_wave1$W1_Relstress1_Gem, "Relatiestress")
# Output: Relatiestress: M = 3.45, SD = 0.82, 95% BI [3.32, 3.58], n = 150

Samenvattend: APA Checklist

  • ✅ Cursief: M, SD, SE, p, t, F, r (niet: n, df, N)
  • ✅ Spaties: M = 3.45 (niet M=3.45)
  • ✅ Decimalen: 2 voor M/SD, 2-3 voor p
  • ✅ Geen leading zero: p = .03, r = .45 (niet 0.03)
  • ✅ p < .001: Bij zeer kleine p-waarden (niet p = .000)
  • ✅ Tabellen: Alleen horizontale lijnen, titel cursief
  • ✅ 95% BI: Altijd rapporteren [ondergrens, bovengrens]
  • ✅ Effect sizes: Cohen's d, η², r bij significante resultaten
