Normaliteit Beoordelen: Scheefheid, Kurtosis en Tests

Veel statistische testen (t-toets, ANOVA, regressie) veronderstellen normale verdeling. Hier leer je hoe je normaliteit beoordeelt en wat je doet bij niet-normale data.

Scheefheid (Skewness)

Scheefheid meet de asymmetrie van een verdeling.

Interpretatie:

  • Scheefheid ‚âà 0: Symmetrisch (normaal)
  • Scheefheid > 0: Rechts-scheef (staart rechts)
  • Scheefheid < 0: Links-scheef (staart links)

Vuistregel: |Scheefheid| < 1 is acceptabel voor normale verdeling

R Code

library(psych)

# Scheefheid berekenen
scheefheid <- psych::skew(data_wave1$W1_Relstress1_Gem, na.rm = TRUE)
print(paste("Scheefheid:", round(scheefheid, 3)))

# Interpretatie
if (abs(scheefheid) < 0.5) {
  print("Vrijwel symmetrisch")
} else if (abs(scheefheid) < 1) {
  print("Licht scheef - acceptabel")
} else {
  print("Sterk scheef - overwegen transformatie")
}

# Histogram voor visualisatie
hist(data_wave1$W1_Relstress1_Gem,
     main = "Verdeling Relatiestress",

     col = "lightblue",

Kurtosis

Kurtosis meet de "gepuntheid" van een verdeling - hoe veel extreme waarden er zijn.

Interpretatie:

  • Kurtosis ‚âà 0: Mesokurtisch (normaal)
  • Kurtosis > 0: Leptokurtisch (gepunt, veel extremen)
  • Kurtosis < 0: Platykurtisch (plat, weinig extremen)

Vuistregel: |Kurtosis| < 1 is acceptabel voor normale verdeling

R Code

# Kurtosis berekenen
kurtosis <- psych::kurtosi(data_wave1$W1_Relstress1_Gem, na.rm = TRUE)
print(paste("Kurtosis:", round(kurtosis, 3)))

# Interpretatie
if (kurtosis > 1) {
  print("Leptokurtisch - veel extreme waarden")
} else if (kurtosis < -1) {
  print("Platykurtisch - weinig extreme waarden")
} else {
  print("Mesokurtisch - normaal")
}

Normaliteitstesten

Shapiro-Wilk Test

De meest gebruikte test voor normaliteit.

Hypothesen:

  • H‚ÇÄ: Data is normaal verdeeld
  • H‚ÇÅ: Data is NIET normaal verdeeld

Beslisregel:

  • p > .05 ‚Üí Normaal verdeeld (H‚ÇÄ behouden)
  • p ‚â§ .05 ‚Üí NIET normaal verdeeld (H‚ÇÄ verwerpen)
# Shapiro-Wilk test
shapiro.test(data_wave1$W1_Relstress1_Gem)

# Voor grotere datasets (n > 5000), gebruik sample
if (nrow(data_wave1) > 5000) {
  sample_data <- sample(data_wave1$W1_Relstress1_Gem, 5000, replace = FALSE)
  shapiro.test(sample_data)
}

‚ö†Ô∏è Waarschuwing: Steekproefgrootte

Bij grote steekproeven (n > 1000) wijst Shapiro-Wilk bijna altijd normaliteit af, zelfs bij triviale afwijkingen!

Oplossing: Kijk naar scheefheid, kurtosis en QQ-plot in plaats van alleen de p-waarde

Q-Q Plot (Quantile-Quantile Plot)

Visuele methode om normaliteit te beoordelen.

# Q-Q Plot maken
qqnorm(data_wave1$W1_Relstress1_Gem,
       main = "Q-Q Plot Relatiestress")
qqline(data_wave1$W1_Relstress1_Gem, col = "red", lwd = 2)

# Interpretatie:
# - Punten op rode lijn = normaal verdeeld
# - Punten boven lijn rechtsboven = rechts-scheef
# - Punten onder lijn linksboven = links-scheef

Oplossingen bij Niet-Normaliteit

Optie 1: Data Transformatie

Veelgebruikte Transformaties:

  • Log-transformatie: Voor rechts-scheve data 
    data$log_inkomen <- log(data$inkomen)
  • Vierkantswortel: Voor gematigde scheefheid 
    data$sqrt_scores <- sqrt(data$scores)
  • Reciproke: Voor extreme scheefheid 
    data$recip_tijd <- 1 / data$reactietijd

Optie 2: Non-Parametrische Testen

Parametrische Test Non-Parametrisch Alternatief
t-toets (2 groepen) Mann-Whitney U test / Wilcoxon
ANOVA (>2 groepen) Kruskal-Wallis test
Pearson correlatie Spearman correlatie

Optie 3: Robuuste Methoden

Gebruik mediaan en IQR in plaats van gemiddelde en SD bij scheefheid.

# In plaats van:
mean(data$inkomen)
sd(data$inkomen)

# Gebruik:
median(data$inkomen)
IQR(data$inkomen)

Complete Normaliteitscheck

normaliteitscheck <- function(x, var_name = "Variable") {
  cat("\n=== Normaliteitscheck:", var_name, "===\n\n")
  
  # 1. Beschrijvende statistieken
  cat("Beschrijvende statistieken:\n")
  cat("N:", sum(!is.na(x)), "\n")
  cat("Gemiddelde:", round(mean(x, na.rm = TRUE), 3), "\n")
  cat("Mediaan:", round(median(x, na.rm = TRUE), 3), "\n")
  cat("SD:", round(sd(x, na.rm = TRUE), 3), "\n\n")
  
  # 2. Scheefheid en kurtosis
  scheef <- psych::skew(x, na.rm = TRUE)
  kurt <- psych::kurtosi(x, na.rm = TRUE)
  cat("Scheefheid:", round(scheef, 3))
  if (abs(scheef) < 0.5) cat(" (vrijwel symmetrisch)\n")
  else if (abs(scheef) < 1) cat(" (licht scheef - acceptabel)\n")
  else cat(" (sterk scheef!)\n")
  
  cat("Kurtosis:", round(kurt, 3))
  if (abs(kurt) < 1) cat(" (acceptabel)\n")
  else cat(" (afwijkend!)\n\n")
  
  # 3. Shapiro-Wilk test
  if (sum(!is.na(x)) <= 5000) {
    shapiro_result <- shapiro.test(x)
    cat("Shapiro-Wilk test: p =", round(shapiro_result$p.value, 4))
    if (shapiro_result$p.value > 0.05) cat(" (normaal)\n")
    else cat(" (NIET normaal)\n")
  } else {
    cat("Sample te groot voor Shapiro-Wilk (n > 5000)\n")
  }
  
  # 4. Visualisaties
  par(mfrow = c(1, 2))
  hist(x, main = paste("Histogram:", var_name),
       xlab = var_name, col = "lightblue", breaks = 20)
  qqnorm(x, main = paste("Q-Q Plot:", var_name))
  qqline(x, col = "red", lwd = 2)
  par(mfrow = c(1, 1))
}

# Gebruik
normaliteitscheck(data_wave1$W1_Relstress1_Gem, "Relatiestress")

Samenvattend

  • Scheefheid: Meet asymmetrie (|scheef| < 1 is acceptabel)
  • Kurtosis: Meet gepuntheid (|kurt| < 1 is acceptabel)
  • Shapiro-Wilk: Test normaliteit (pas op bij grote n!)
  • Q-Q Plot: Visuele beoordeling (punten op lijn = normaal)
  • Niet-normaal? Transformeer, gebruik non-parametrisch, of mediaan/IQR
