Van SPSS naar R: Beschrijvende Statistieken

Voor SPSS gebruikers: hoe doe je dezelfde analyses in R? Deze gids toont equivalente commando's en waarom R vaak beter is.

Analyze ‚Üí Descriptive Statistics ‚Üí Descriptives

SPSS Menu

SPSS stappen:

  1. Analyze ‚Üí Descriptive Statistics ‚Üí Descriptives
  2. Selecteer variabelen ‚Üí pijltje naar rechts
  3. Options: Mean, Std. deviation, Minimum, Maximum
  4. OK

R Equivalent

library(psych)

# E√©n variabele
describe(data$variabele)

# Meerdere variabelen (zoals SPSS)
describe(data[, c("var1", "var2", "var3")])

# Of met basis R
summary(data$variabele)

# Custom statistieken kiezen
library(dplyr)
data %>%
  summarise(
    N = n(),
    Mean = mean(variabele, na.rm = TRUE),
    SD = sd(variabele, na.rm = TRUE),
    Min = min(variabele, na.rm = TRUE),
    Max = max(variabele, na.rm = TRUE)
  )

Analyze ‚Üí Descriptive Statistics ‚Üí Frequencies

SPSS Menu

SPSS stappen:

  1. Analyze ‚Üí Descriptive Statistics ‚Üí Frequencies
  2. Selecteer variabelen
  3. Statistics: Quartiles, Mean, Median, Mode, Std. deviation
  4. Charts: Histograms
  5. OK

R Equivalent

# Frequentietabel
table(data$variabele)

# Met percentages
prop.table(table(data$variabele)) * 100

# Complete output (zoals SPSS Frequencies)
library(psych)
describe(data$variabele)

# Histogram
hist(data$variabele,
     main = "Histogram",
     xlab = "Variabele",
     col = "lightblue",
     breaks = 20)

# Of met ggplot2 (mooier)
library(ggplot2)
ggplot(data, aes(x = variabele)) +
  geom_histogram(fill = "#667eea", color = "white", bins = 20) +
  labs(title = "Histogram", x = "Variabele", y = "Frequentie") +
  theme_minimal()

Analyze ‚Üí Descriptive Statistics ‚Üí Explore

SPSS Menu

SPSS stappen:

  1. Analyze ‚Üí Descriptive Statistics ‚Üí Explore
  2. Dependent List: selecteer variabelen
  3. Factor List: groepsvariabele (optioneel)
  4. Statistics: Descriptives, Outliers, Percentiles
  5. Plots: Boxplot, Histogram, Normality plots with tests
  6. OK

R Equivalent

library(psych)
library(ggplot2)

# Volledige exploratie (zoals SPSS Explore)
describe(data$variabele)

# Normaliteitstesten
shapiro.test(data$variabele)

# Q-Q plot
qqnorm(data$variabele)
qqline(data$variabele, col = "red")

# Boxplot
boxplot(data$variabele,
        main = "Boxplot",
        ylab = "Variabele",
        col = "lightblue")

# Histogram met normale curve
hist(data$variabele,
     probability = TRUE,
     col = "lightblue",
     main = "Histogram met Normale Curve")
curve(dnorm(x, mean = mean(data$variabele, na.rm = TRUE),
            sd = sd(data$variabele, na.rm = TRUE)),
      add = TRUE, col = "red", lwd = 2)

# Uitschieters detecteren (zoals SPSS)
outliers <- boxplot.stats(data$variabele)$out
cat("Uitschieters:", outliers)

Per Groep (zoals Factor List in SPSS)

# Beschrijvende statistieken per groep
library(dplyr)
data %>%
  group_by(geslacht) %>%
  summarise(
    N = n(),
    Mean = mean(variabele, na.rm = TRUE),
    SD = sd(variabele, na.rm = TRUE),
    Median = median(variabele, na.rm = TRUE)
  )

# Boxplot per groep
ggplot(data, aes(x = geslacht, y = variabele, fill = geslacht)) +
  geom_boxplot() +
  labs(title = "Boxplot per Groep") +
  theme_minimal()

# BETER: skimr voor volledige exploratie
library(skimr)
skim(data)

Waarom R Beter is dan SPSS

1. Reproduceerbaarheid

SPSS: Klik-gedrag niet reproduceerbaar, syntax soms vergeten op te slaan

R: Alles in scripts ‚Üí perfecte reproduceerbaarheid

2. Automatisering

# R: Beschrijvende statistieken voor 50 variabelen in 1 regel
library(psych)
describe(data[, 1:50])

# SPSS: 50 keer menu's klikken OF lange syntax typen

3. Flexibiliteit

SPSS: Beperkt tot vooraf gedefinieerde opties

R: Onbeperkte custom functies en packages

4. Visualisaties

# R: Professionele plots met ggplot2
library(ggplot2)
ggplot(data, aes(x = geslacht, y = stress, fill = geslacht)) +
  geom_violin(alpha = 0.5) +
  geom_boxplot(width = 0.2) +
  geom_jitter(alpha = 0.3, width = 0.1) +
  labs(title = "Stress per Geslacht",
       y = "Stress Score") +
  theme_minimal() +
  scale_fill_manual(values = c("#667eea", "#f093fb"))

# SPSS: Basis plots, moeilijk aan te passen

  ))

Transitie Tips voor SPSS Gebruikers

Start Hier:

  1. Install RStudio: Gebruiksvriendelijke interface (zoals SPSS)
  2. Learn psych::describe(): Vervangt 90% van SPSS Descriptives/Explore
  3. Master dplyr: Voor groepsgewijze analyses (SPLIT FILE in SPSS)
  4. Use RMarkdown: Combineer analyses + rapportage (beter dan SPSS Output Viewer)
  5. Embrace scripting: Alles in code = reproduceerbaar en deelbaar

Taak: Beschrijvende statistieken per geslacht voor 3 stressmaten

SPSS Workflow:

  1. Data ‚Üí Split File ‚Üí Organize output by groups ‚Üí Select geslacht
  2. Analyze ‚Üí Descriptive Statistics ‚Üí Descriptives
  3. Selecteer stress1, stress2, stress3
  4. Options ‚Üí Mean, SD, Min, Max

  6. Copy output naar Word

R Workflow:

library(dplyr)
library(knitr)

# Analyse + Tabel in 1 code block
results <- data %>%
  group_by(geslacht) %>%
  summarise(across(
    c(stress1, stress2, stress3),
    list(M = ~mean(., na.rm = TRUE), 
         SD = ~sd(., na.rm = TRUE),
         Min = ~min(., na.rm = TRUE),
         Max = ~max(., na.rm = TRUE))
  ))

# Direct naar professionele tabel
kable(results, digits = 2, caption = "Beschrijvende Statistieken per Geslacht")
