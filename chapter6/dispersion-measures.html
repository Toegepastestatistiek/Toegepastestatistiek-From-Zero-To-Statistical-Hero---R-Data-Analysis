Spreidingsmaten: Bereik, IQR, SD en Variantie

Spreidingsmaten geven aan hoe verspreid de data zijn. Ze beantwoorden de vraag: "Liggen alle waarden dicht bij elkaar of wijd uiteen?"

Waarom Spreiding Belangrijk Is

Twee datasets kunnen hetzelfde gemiddelde hebben, maar heel verschillende spreiding:

  • Groep A: Werkstress scores: 3.0, 3.1, 3.2, 2.9, 3.0 → x̄ = 3.04, SD = 0.11 (weinig spreiding)
  • Groep B: Werkstress scores: 1.0, 2.5, 3.0, 4.0, 4.7 → x̄ = 3.04, SD = 1.45 (veel spreiding)

Conclusie: Gemiddelde alleen vertelt niet het hele verhaal!

Bereik (Range)

Definitie

Het bereik is het verschil tussen de hoogste en laagste waarde in de dataset.

Formule: Bereik = Maximum - Minimum

Symbool: R of Range

Meetniveau: Ordinaal, Metrisch

Voorbeelden

Voorbeeld 1: Opleidingsniveau (Ordinaal)

Data: 1 (Laag), 1, 2 (Midden), 2, 3 (Hoog), 3, 3

  • Minimum = 1
  • Maximum = 3
  • Bereik = 3 - 1 = 2

Voorbeeld 2: Werkstress (Metrisch, schaal 1-5)

Data: 1.5, 2.0, 2.5, 3.0, 4.5

  • Minimum = 1.5
  • Maximum = 4.5
  • Bereik = 4.5 - 1.5 = 3.0

R Code

# Bereik berekenen
minimum <- min(data_wave1$W1_Diploma, na.rm = TRUE)
maximum <- max(data_wave1$W1_Diploma, na.rm = TRUE)
bereik <- maximum - minimum

# Of in één regel
bereik <- diff(range(data_wave1$W1_Diploma, na.rm = TRUE))

# Met dplyr
data_wave1 %>%
  summarise(
    Min = min(W1_Diploma, na.rm = TRUE),
    Max = max(W1_Diploma, na.rm = TRUE),
    Bereik = Max - Min
  )

Nadelen van het Bereik

⚠️ Zeer Gevoelig voor Uitschieters!

Probleem: Één extreme waarde verandert het hele bereik drastisch.

Voorbeeld:

  • Zonder uitschieter: 2, 3, 3, 4, 4 → Bereik = 4-2 = 2
  • Met uitschieter: 2, 3, 3, 4, 99 → Bereik = 99-2 = 97 (!)

Oplossing: Gebruik interkwartielafstand (IQR) voor robuuste spreiding

Interkwartielafstand (IQR)

Definitie

De interkwartielafstand (Interquartile Range, IQR) is het verschil tussen het derde kwartiel (Q3) en het eerste kwartiel (Q1). Dit meet de spreiding van de middelste 50% van de data.

Formule: IQR = Q3 - Q1

Symbool: IQR of IQA (Interkwartielafstand)

Meetniveau: Ordinaal, Metrisch

Visualisatie met Boxplot

    Maximum (of Q3 + 1.5×IQR)
         │
         ┼───────┐
         │       │
    Q3   ┼───────┤
         │       │  ← Middelste 50% (IQR)
         │   ●   │  ← Mediaan (Q2)
         │       │
    Q1   ┼───────┤
         │       │
         ┼───────┘
         │
    Minimum (of Q1 - 1.5×IQR)

IQR bevat de middelste 50% van alle observaties

Voorbeelden

Voorbeeld: 13 werkstress scores

Data (gesorteerd): 1.5, 2.0, 2.2, 2.5, 2.8, 3.0, 3.2, 3.5, 3.8, 4.0, 4.2, 4.5, 4.8

  • Q1 (positie 3.5): (2.2 + 2.5) / 2 = 2.35
  • Q2/Mediaan (positie 7): 3.2
  • Q3 (positie 10.5): (4.0 + 4.2) / 2 = 4.1
  • IQR = 4.1 - 2.35 = 1.75

R Code

# IQR direct berekenen
iqr <- IQR(data_wave1$W1_Diploma, na.rm = TRUE)

# Handmatige berekening
Q1 <- quantile(data_wave1$W1_Diploma, 0.25, na.rm = TRUE)
Q3 <- quantile(data_wave1$W1_Diploma, 0.75, na.rm = TRUE)
iqr_handmatig <- Q3 - Q1

# Met summary() voor volledig overzicht
summary(data_wave1$W1_Diploma)

# Boxplot maken
boxplot(data_wave1$W1_Diploma,
        main = "Verdeling Opleidingsniveau",
        ylab = "Niveau",
        col = "lightblue")

Uitschieters Detecteren met IQR

1.5 × IQR Regel (Tukey's Fence)

Een observatie is een uitschieter als:

  • Lage uitschieter: Waarde < Q1 - 1.5×IQR
  • Hoge uitschieter: Waarde > Q3 + 1.5×IQR

Voorbeeld Berekening

Stel: Q1 = 2.35, Q3 = 4.1, IQR = 1.75

  • Ondergrens = 2.35 - 1.5×1.75 = 2.35 - 2.625 = -0.275
  • Bovengrens = 4.1 + 1.5×1.75 = 4.1 + 2.625 = 6.725

Waarden < -0.275 of > 6.725 zijn uitschieters

R Code: Uitschieters Detecteren

# Uitschieters identificeren
Q1 <- quantile(data_wave1$W1_Relstress1_Gem, 0.25, na.rm = TRUE)
Q3 <- quantile(data_wave1$W1_Relstress1_Gem, 0.75, na.rm = TRUE)
IQR_val <- Q3 - Q1

# Grenzen berekenen
ondergrens <- Q1 - 1.5 * IQR_val
bovengrens <- Q3 + 1.5 * IQR_val

# Uitschieters vinden
uitschieters <- data_wave1 %>%
  filter(W1_Relstress1_Gem < ondergrens | 
         W1_Relstress1_Gem > bovengrens)

cat("Aantal uitschieters:", nrow(uitschieters), "\n")
cat("Ondergrens:", round(ondergrens, 2), "\n")
cat("Bovengrens:", round(bovengrens, 2), "\n")

Bereik vs. IQR

Eigenschap Bereik IQR
Formule Max - Min Q3 - Q1
Meet 100% van data Middelste 50%
Gevoelig voor uitschieters Ja (zeer!) Nee (robuust)
Gebruik Snelle indruk Betrouwbare spreiding

Standaardafwijking (SD)

Definitie

De standaardafwijking meet de gemiddelde afwijking van elke observatie ten opzichte van het gemiddelde. Het is de wortel van de variantie.

Steekproef SD:

s = √[ Σ(xi - x̄)² / (n-1) ]

Symbool: s (steekproef) of σ (populatie) of SD

Meetniveau: Metrisch ALLEEN

Eenheid: Zelfde als originele data (jaren, euro's, punten)

Stapsgewijze Berekening

Voorbeeld: Werkstress van 5 respondenten

Scores: 2, 3, 3, 4, 4

xi xi - x̄ (xi - x̄)²
2 2 - 3.2 = -1.2 1.44
3 3 - 3.2 = -0.2 0.04
3 3 - 3.2 = -0.2 0.04
4 4 - 3.2 = 0.8 0.64
4 4 - 3.2 = 0.8 0.64
Σ = 16 Σ = 0 Σ = 2.80

Stap 1: Gemiddelde: x̄ = 16/5 = 3.2

Stap 2: Variantie: s² = 2.80 / (5-1) = 2.80 / 4 = 0.70

Stap 3: SD: s = √0.70 = 0.84

Waarom n-1? (Bessel's Correctie)

Vraag: Waarom delen door (n-1) en niet door n?

Antwoord: We gebruiken n-1 omdat we het steekproefgemiddelde (x̄) gebruiken in plaats van het populatiegemiddelde (μ).

  • Populatie SD: σ = √[Σ(x-μ)² / N] (delen door N)
  • Steekproef SD: s = √[Σ(x-x̄)² / (n-1)] (delen door n-1)

Reden: Door x̄ te gebruiken onderschat je de variabiliteit. Delen door (n-1) corrigeert dit.

R Code

# Standaardafwijking berekenen
sd_waarde <- sd(data_wave1$W1_Relstress1_Gem, na.rm = TRUE)
print(paste("SD:", round(sd_waarde, 2)))

# Handmatige berekening (ter verificatie)
gemiddelde <- mean(data_wave1$W1_Relstress1_Gem, na.rm = TRUE)
afwijkingen <- data_wave1$W1_Relstress1_Gem - gemiddelde
gekwadrateerde_afwijkingen <- afwijkingen^2
variantie <- sum(gekwadrateerde_afwijkingen, na.rm = TRUE) / 
             (sum(!is.na(afwijkingen)) - 1)
sd_handmatig <- sqrt(variantie)

# Controleer
all.equal(sd_waarde, sd_handmatig)

SD Interpreteren: 68-95-99.7 Regel

Bij een normale verdeling geldt:

  • 68% van data ligt binnen x̄ ± 1 SD
  • 95% van data ligt binnen x̄ ± 2 SD
  • 99.7% van data ligt binnen x̄ ± 3 SD

Voorbeeld: IQ-scores

x̄ = 100, SD = 15

  • 68% heeft IQ tussen 85 en 115 (100 ± 15)
  • 95% heeft IQ tussen 70 en 130 (100 ± 30)
  • 99.7% heeft IQ tussen 55 en 145 (100 ± 45)

Variantie

Definitie

De variantie is het kwadraat van de standaardafwijking. Het meet de spreiding in gekwadrateerde eenheden.

Formule: Variantie = SD²

s² = Σ(xi - x̄)² / (n-1)

Symbool: s² (steekproef) of σ² (populatie)

Eenheid: Gekwadrateerde originele eenheden (jaren², euro²)

Relatie SD en Variantie

Van Naar Formule
Variantie SD SD = √Variantie
SD Variantie Variantie = SD²

R Code

# Variantie berekenen
var_waarde <- var(data_wave1$W1_Relstress1_Gem, na.rm = TRUE)
sd_waarde <- sd(data_wave1$W1_Relstress1_Gem, na.rm = TRUE)

# Controleer relatie
var_waarde == sd_waarde^2  # Should be TRUE
sd_waarde == sqrt(var_waarde)  # Should be TRUE

# Rapporteer beide
cat("Gemiddelde:", round(mean(data_wave1$W1_Relstress1_Gem, na.rm = TRUE), 2), "\n")
cat("SD:", round(sd_waarde, 2), "\n")
cat("Variantie:", round(var_waarde, 2), "\n")

Wanneer Variantie vs. SD?

Gebruik STANDAARDAFWIJKING (SD) voor:

  • Beschrijvende statistiek rapporteren
  • Interpretatie (zelfde eenheid als data)
  • APA-rapportage: M = 3.2, SD = 0.84

Gebruik VARIANTIE voor:

  • ANOVA (variance analysis)
  • Regressie berekeningen
  • Theoretische formules
  • Wiskundige afleidingen

Vergelijking Alle Spreidingsmaten

Maat Formule Meetniveau Robuust? Gebruik
Bereik Max - Min Ordinaal, Metrisch Nee Snelle indruk
IQR Q3 - Q1 Ordinaal, Metrisch Ja Robuuste spreiding, uitschieters detecteren
SD √[Σ(x-x̄)²/(n-1)] Metrisch Nee Standaard rapportage, interpretatie
Variantie Σ(x-x̄)²/(n-1) Metrisch Nee ANOVA, regressie, berekeningen

Veelgemaakte Fouten

❌ Fout 1: SD van ordinale variabelen

Verkeerd: sd(Tevredenheid) waar 1=Ontevreden, 2=Neutraal, 3=Tevreden

Waarom fout? SD vereist metrisch meetniveau (gelijke afstanden)

Correct: Gebruik IQR voor ordinale variabelen

❌ Fout 2: Variantie interpreteren

Verkeerd: "Gemiddelde werkstress = 3.2 (Variantie = 0.70)"

Waarom misleidend? Variantie in gekwadrateerde eenheden - niet direct interpreteerbaar

Correct: "Gemiddelde = 3.2 (SD = 0.84)" - zelfde eenheid als data

❌ Fout 3: Bereik bij uitschieters

Verkeerd: Alleen bereik rapporteren bij scheve verdelingen

Waarom problematisch? Één uitschieter geeft vertekend beeld van spreiding

Correct: Rapporteer IQR of verwijder uitschieters eerst

R Code: Alle Spreidingsmaten Tegelijk

# Optie 1: Via psych::describe()
library(psych)
psych::describe(data_wave1$W1_Relstress1_Gem)

# Optie 2: Handmatig alle maten
data_wave1 %>%
  summarise(
    N = n(),
    Min = min(W1_Relstress1_Gem, na.rm = TRUE),
    Q1 = quantile(W1_Relstress1_Gem, 0.25, na.rm = TRUE),
    Mediaan = median(W1_Relstress1_Gem, na.rm = TRUE),
    Q3 = quantile(W1_Relstress1_Gem, 0.75, na.rm = TRUE),
    Max = max(W1_Relstress1_Gem, na.rm = TRUE),
    Bereik = Max - Min,
    IQR = Q3 - Q1,
    Gemiddelde = mean(W1_Relstress1_Gem, na.rm = TRUE),
    SD = sd(W1_Relstress1_Gem, na.rm = TRUE),
    Variantie = var(W1_Relstress1_Gem, na.rm = TRUE)
  )

Samenvattend

  • Bereik: Max - Min, eenvoudig maar gevoelig voor uitschieters
  • IQR: Q3 - Q1, robuust alternatief voor bereik
  • SD: Gemiddelde afwijking van x̄, meest gebruikte spreidingsmaat
  • Variantie: SD², gebruikt in ANOVA en regressie
  • Keuze hangt af van: Meetniveau, uitschieters, doel analyse
